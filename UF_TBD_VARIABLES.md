# UF_TBD_VARIABLES.md

Переменные-заглушки, требующие замены на реальные UF_CRM коды (или другой стандартный источник Bitrix24):

| Переменная | Смысл | Рекомендованное место хранения |
|---|---|---|
| `{UF_CRM__TBD__CITY__}` | Город в шапке | Сделка/Смарт-процесс (UF «Город документа») |
| `{UF_CRM__TBD__SUPPLIER_SIGNER_FIO__}` | Подписант Поставщика (ФИО полностью) | Сделка/Смарт-процесс |
| `{UF_CRM__TBD__SUPPLIER_SIGNER_BASIS__}` | Основание полномочий Поставщика | Сделка/Смарт-процесс |
| `{UF_CRM__TBD__SUPPLIER_SIGNER_SHORT__}` | Подпись Поставщика (инициалы/фамилия) | Сделка/Смарт-процесс |
| `{UF_CRM__TBD__BUYER_GENDER_ENDING__}` | Окончание для «именуем…» | Сделка/Смарт-процесс (опционально) |
| `{UF_CRM__TBD__BUYER_SIGNER_FIO__}` | Подписант Покупателя (ФИО) | Сделка/Смарт-процесс |
| `{UF_CRM__TBD__BUYER_SIGNER_BASIS__}` | Основание полномочий Покупателя | Сделка/Смарт-процесс |
| `{UF_CRM__TBD__BUYER_SIGNER_POST__}` | Должность подписанта Покупателя | Сделка/Смарт-процесс |
| `{UF_CRM__TBD__BUYER_SIGNER_SHORT__}` | Подпись Покупателя (инициалы/фамилия) | Сделка/Смарт-процесс |
| `{UF_CRM__TBD__BUYER_PHONE__}` | Телефон Покупателя (если не тянется из контактов компании) | Компания/Сделка |
| `{UF_CRM__TBD__BUYER_EMAIL__}` | E-mail Покупателя (если не тянется из контактов компании) | Компания/Сделка |
| `{UF_CRM__TBD__CONTRACT_END_YEAR__}` | Год окончания срока договора | Сделка/Смарт-процесс |
