# VARIABLE_MAP.md

| Переменная | Где хранится | Обязательна | Комментарий |
|---|---|---|---|
| `{DocumentNumber}` | Документ CRM | Да | Номер договора/документа |
| `{DocumentCreateTime~d.m.Y}` | Документ CRM | Да | Дата документа в шапке |
| `{DocumentCreateTime~Y}` | Документ CRM | Нет | Год; если не поддерживается — заменить на UF или удалить вручную |
| `{UF_CRM__TBD__CITY__}` | Сделка/Смарт-процесс | Да | Город в шапке (если нужен по шаблону) |
| `{MyCompanyRequisiteRqCompanyName}` | Моя компания → Реквизиты | Да | Поставщик: наименование |
| `{MyCompanyRequisiteRegisteredAddressText}` | Моя компания → Адрес | Да | Поставщик: адрес |
| `{MyCompanyRequisiteRqOgrn}` | Моя компания → Реквизиты | Да | Поставщик: ОГРН |
| `{MyCompanyRequisiteRqInn}` | Моя компания → Реквизиты | Да | Поставщик: ИНН |
| `{MyCompanyRequisiteRqKpp}` | Моя компания → Реквизиты | Да | Поставщик: КПП |
| `{MyCompanyBankDetailRqAccNum}` | Моя компания → Банк-реквизиты | Да | Поставщик: р/с |
| `{MyCompanyBankDetailRqBankName}` | Моя компания → Банк-реквизиты | Да | Поставщик: банк |
| `{MyCompanyBankDetailRqCorrAccNum}` | Моя компания → Банк-реквизиты | Да | Поставщик: к/с (проверь наличие поля в портале) |
| `{MyCompanyBankDetailRqBik}` | Моя компания → Банк-реквизиты | Да | Поставщик: БИК |
| `{MyCompanyPhone}` | Моя компания → Контакты | Да | Телефон |
| `{MyCompanyEmail}` | Моя компания → Контакты | Да | E-mail |
| `{RequisiteRqCompanyName}` | Компания (контрагент) → Реквизиты | Да | Покупатель: наименование |
| `{RequisiteRegisteredAddressText}` | Компания (контрагент) → Реквизиты/Адрес | Да | Покупатель: адрес |
| `{RequisiteRqOgrn}` | Компания (контрагент) → Реквизиты | Да/Нет | Для ИП будет пусто; тогда нужен ОГРНИП в UF |
| `{RequisiteRqInn}` | Компания (контрагент) → Реквизиты | Да | ИНН |
| `{RequisiteRqKpp}` | Компания (контрагент) → Реквизиты | Да/Нет | У ИП нет КПП |
| `{BankDetailRqAccNum}` | Компания (контрагент) → Банк-реквизиты | Да | Покупатель: р/с |
| `{BankDetailRqBankName}` | Компания (контрагент) → Банк-реквизиты | Да | Покупатель: банк |
| `{BankDetailRqCorrAccNum}` | Компания (контрагент) → Банк-реквизиты | Да | Покупатель: к/с (проверь наличие поля) |
| `{BankDetailRqBic}` | Компания (контрагент) → Банк-реквизиты | Да | Покупатель: БИК |
| `{UF_CRM__TBD__SUPPLIER_SIGNER_FIO__}` | Сделка/Смарт-процесс | Да | Подписант Поставщика (ФИО полностью) |
| `{UF_CRM__TBD__SUPPLIER_SIGNER_BASIS__}` | Сделка/Смарт-процесс | Да | Основание полномочий Поставщика |
| `{UF_CRM__TBD__SUPPLIER_SIGNER_SHORT__}` | Сделка/Смарт-процесс | Да | Подпись: инициалы/фамилия (как в исходнике) |
| `{UF_CRM__TBD__BUYER_SIGNER_FIO__}` | Сделка/Смарт-процесс | Да | Подписант Покупателя (ФИО) |
| `{UF_CRM__TBD__BUYER_SIGNER_BASIS__}` | Сделка/Смарт-процесс | Да | Основание полномочий Покупателя |
| `{UF_CRM__TBD__BUYER_SIGNER_POST__}` | Сделка/Смарт-процесс | Да | Должность подписанта |
| `{UF_CRM__TBD__BUYER_SIGNER_SHORT__}` | Сделка/Смарт-процесс | Да | Подпись: инициалы/фамилия |
| `{UF_CRM__TBD__BUYER_PHONE__}` | Сделка/Компания | Желательно | В исходнике пусто; лучше тянуть из компании или UF |
| `{UF_CRM__TBD__BUYER_EMAIL__}` | Сделка/Компания | Желательно | В исходнике пусто; лучше тянуть из компании или UF |
| `{UF_CRM__TBD__CONTRACT_END_YEAR__}` | Сделка/Смарт-процесс | Да | В исходнике 2025; если шаблон многоразовый — вынести в UF |
| `{UF_CRM__TBD__BUYER_GENDER_ENDING__}` | Сделка/Смарт-процесс | Нет | Чтобы корректно склонять “именуемый/именуемое”; можно убрать и фиксировать “именуемое” |
